✨ Features

🎡 Animated spin wheel with multicolor segments
➕ Add custom tasks/options
❌ Delete tasks easily
💡 Blinking casino lights around the wheel
🏆 Winner popup after each spin
💾 Tasks saved locally (persists between sessions)
🎰 Retro casino theme — red, gold & brown


🛠️ Tech Stack

React Native + Expo (SDK 56)
TypeScript
Zustand (state management)
AsyncStorage (local persistence)
React Native Reanimated (spin animation)
React Native SVG (wheel rendering)
